# BBS::Universal Configuration

![BBS::Universal Logo](../files/files/BBS/BBS_Universal.png?raw=true "BBS::Universal")

Change the ```bbs.rc``` file to reflect your database information.

## Valid Location

* ```./conf/bbs.rc```
* ```~/.bbs_universal/bbs.rc```
* ```/etc/bbs.rc```

The paths are in the order searched.  Only the first one found will be loaded.
